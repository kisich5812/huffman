# Алгоритм Хаффмана

Программа реализует алгоритм сжатия Хаффмана. Программа работает для текстовых файлов написанных с использованием английских букв, цифр и специальных символов.

Сборка:
```
$ git clone https://github.com/kisich5812/huffman/
$ cd huffman && make
```

Программа имеет следующие опции:

-o <output_file>	Указать имя выходного файла(необязательная опция)

-h, --help		Вывести эту справку

--screen		Вывести таблицу символов на экран(не отключает вывод в файл)

-i <input_file>		Файл, данные из которого будут зашифрованы

-e, --enc		Зашифровать данные

-d, --dec		Дешифровать данные

-t <table_file>		Файл для записи кодовой таблицы(необязательная опция в случае шифрования)

Пример использования:

```
$ ./huffman -i ex1.txt --enc
$ ./huffman -i output.huf -t table.huf -o output.txt --dec
```

Первая команда сожмёт файл ex1.txt, так как не указаны имена для файлов с таблицей кодов и выходного файла, они будут названы `table.huf` и `output.huf` соответственно.

Вторая команда расшифрует файл `output.huf` используя данные из таблицы `table.huf`.
